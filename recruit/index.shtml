サークルでも

学生団体でもない

私たちは高知工科大学情報学群直属のプロジェクトチームです。
メンバーは全て情報学群生で構成されており、当Webサイトの設計および運営を行なっています。
コンテンツ制作を行う時に必要なページのデザインや設計、それに関わる運営など支えあいながらWebサイトを作り上げています。

未経験者でも問題ない

Webに興味はあるけど技術がないから不安なんて心配は持たなくて大丈夫です。なぜならばメンバーのほとんどが最初は未経験だったからです。
最初は先輩メンバーに教えてもらいながら少しずつスキルアップできる環境があります。

まずはできそうなことから参加していき、少しずつ慣れていきましょう。初めからすべてできる人はそうそう居ないですよね。
慣れてきたらもっと応用的な部分を担当していき自身のスキルアップを体感しましょう。大丈夫、みんなが支えてくれます。

  • Web Designer

    Webサイトにおけるデザインを担当します。UI/UXデザインやバナーのデザインなど、ユーザビリティの向上にダイレクトに繋がるのがデザイナーです。

  • Coding

    HTMLやCSSなどをはじめとしたフロントエンドから、サーバ管理やCMS管理などのバックエンドなど、個人のやりたい分野に応じて担当します。

  • Photographer

    記事に必要な写真やコンテンツ制作における撮影などを行います。

  • Writer

    自ら情報を取ってきたり、取材などを行いながらニュース記事作成を行います。

せっかく情報学群に入学したのなら挑戦してみませんか？初心者でも何も問題ありません！何故なら初心者が集まってこのサイトを作ったのですから。
